# bookmarkSearch-uToolsPlugin

#### 浏览器书签搜索

默认支持Chrome浏览器书签搜索，如使用其他Chromium内核浏览器，请键入`setBookmarksPath`修改书签文件路径，暂时只同时支持单个浏览器书签

**搜索书签后 `↑` `↓` 选择，`→` 、 `回车` 或者鼠标点击 调用默认浏览器打开书签链接**
